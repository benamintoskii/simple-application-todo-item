# LimeStore - MEAN Stack Product Showcase

A modern and responsive MEAN Stack web application built using Node.js, Express.js, and Angular.
The project displays a curated collection of 9 products in a clean and visually appealing interface with smooth hover animations and a modern lime-green design theme.

The backend provides a simple REST API using a hardcoded in-memory data structure, while the Angular frontend fetches and displays the products dynamically.

---

# Features

* Responsive modern UI
* Express.js REST API
* Angular frontend integration
* Smooth hover animations
* Animated product image gallery
* Clean lime-green theme
* Responsive card layout
* Modern typography and spacing
* Real product images from public sources

# Backend Setup

Navigate to the backend folder:

```bash id="fjlwmz"
cd backend
```

Install dependencies:

```bash id="9qj9rw"
npm install
```

Start the backend server:

```bash id="h8r0c9"
npm start
```

Backend API runs on:

```bash id="qbzhyx"

```
